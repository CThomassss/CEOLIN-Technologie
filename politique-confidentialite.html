Politique de Confidentialité

1. Introduction

CEOLIN Technologies s'engage à protéger la confidentialité de vos données personnelles. Cette politique explique comment nous collectons, utilisons et protégeons vos informations.

En utilisant notre site web, vous acceptez les pratiques décrites dans cette politique de confidentialité.

2. Données collectées

Nous collectons les informations suivantes :

  • Informations de contact : nom, adresse email, numéro de téléphone (via le formulaire de contact)
  • Données de navigation : adresse IP, type de navigateur, pages visitées
  • Communications : messages envoyés via le formulaire de contact

3. Utilisation des données

Vos données personnelles sont utilisées pour :

  • Répondre à vos demandes et questions
  • Vous contacter concernant nos services
  • Améliorer notre site web et nos services
  • Respecter nos obligations légales

4. Base légale du traitement

Conformément au RGPD, le traitement de vos données personnelles repose sur :

  • Consentement : pour l'envoi de communications commerciales
  • Intérêt légitime : pour répondre à vos demandes et améliorer nos services
  • Exécution contractuelle : pour la fourniture de nos services

5. Conservation des données

Nous conservons vos données personnelles pendant :

  • Demandes de contact : 3 ans après le dernier contact
  • Prospects : 3 ans après le dernier contact
  • Clients : 5 ans après la fin de la relation contractuelle

6. Partage des données

Nous ne vendons, ne louons, ni ne partageons vos données personnelles avec des tiers, sauf :

  • Avec votre consentement explicite
  • Pour respecter une obligation légale
  • Avec nos prestataires techniques (hébergement, email) sous contrat de confidentialité

7. Sécurité des données

Nous mettons en œuvre des mesures de sécurité appropriées :

  • Chiffrement SSL/TLS pour les transmissions
  • Accès restreint aux données personnelles
  • Sauvegardes régulières et sécurisées
  • Mise à jour régulière de nos systèmes de sécurité

8. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : connaître les données que nous détenons sur vous
  • Droit de rectification : corriger les données inexactes
  • Droit à l'effacement : supprimer vos données personnelles
  • Droit à la portabilité : récupérer vos données dans un format structuré
  • Droit d'opposition : vous opposer au traitement de vos données
  • Droit de limitation : limiter le traitement de vos données

Pour exercer ces droits, contactez-nous à : contact@ceolin-technologies.com

9. Cookies

Notre site utilise des cookies pour :

  • Améliorer le fonctionnement du site
  • Analyser la fréquentation (Google Analytics - anonymisé)
  • Personnaliser votre expérience

Vous pouvez désactiver les cookies dans les paramètres de votre navigateur.

10. Modifications

Cette politique de confidentialité peut être modifiée. Nous vous informerons de tout changement significatif par email ou via notre site web.

Dernière mise à jour : Octobre 2025

Pour toute question concernant cette politique de confidentialité :
